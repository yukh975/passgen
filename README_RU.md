# Secure Password Generator v1.0.0

[![JavaScript](https://img.shields.io/badge/JavaScript-ES2020-f7df1e?logo=javascript&logoColor=black)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE.md)

**Лёгкий полностью клиентский генератор паролей с тремя режимами: Пароль, Фраза-пароль и PIN. Вся генерация происходит в браузере через Web Crypto API — никакие данные не передаются на сервер.**

---

## Демо

Текущая версия доступна по адресу **https://yukh.net/passgen/**

---

## Возможности

### Пароль
- Настраиваемая длина (4–64 символа, по умолчанию 20)
- Наборы символов: заглавные, строчные, цифры, специальные символы
- Исключение визуально похожих символов (I, l, 1, O, 0)
- Запрет повторяющихся символов
- Поля для дополнительных и исключаемых символов
- **8 пресетов для платформ**: PostgreSQL, MySQL, SQL Server, Linux User, Windows Admin, Redis, RabbitMQ, High Entropy
- Индикатор надёжности на основе энтропии Шеннона и разнообразия символов — **Очень сильный** требует ≥ 80 бит и 3+ типа символов

### Фраза-пароль
- Случайные наборы слов из встроенного словаря ~500 слов
- Настраиваемое количество слов (2–10, по умолчанию 8)
- Варианты регистра: с заглавной буквы или случайный для каждой буквы
- Опциональная цифра в начале и/или конце (приклеивается к слову без разделителя)
- Числа внутри фразы (случайно добавляет цифру к части слов)
- Разделитель: пробел, дефис (по умолчанию), точка, подчёркивание, случайный символ или без разделителя

### PIN
- Числовой код, настраиваемая длина (4–12 цифр, по умолчанию 6)

### Общее
- Генерация от 1 до 50 результатов за раз на всех трёх вкладках
- Кнопка копирования у каждого результата; **Скопировать все** открывает модальное окно с textarea
- Кнопка «Пересоздать» — общая для всех результатов на вкладке
- Переключатель тёмной/светлой темы, переключатель языка EN / RU
- Активная вкладка и все настройки генератора сохраняются в `localStorage`; кнопка **сброса** (↺) возвращает все настройки по умолчанию
- Криптографически стойкая случайность через `window.crypto.getRandomValues()` с rejection sampling

## Установка

```bash
git clone https://github.com/yukh975/passgen.git
cd passgen
```

Откройте `index.html` в браузере — сборка не требуется, зависимостей нет.

Или запустите любой статический сервер:

```bash
npx serve .
```

## Безопасность

- Вся случайность берётся из `window.crypto.getRandomValues()` с rejection sampling для устранения смещения по модулю.
- Данные не покидают браузер — генерация полностью локальная.
- Нет внешних CDN, удалённых ресурсов и трекинга.

# История изменений

## v1.0.0 — не выпущено

### Интерфейс
- Кнопка переключения тёмной/светлой темы (иконка солнца/луны) в заголовке; по умолчанию тёмная тема; выбор сохраняется в `localStorage`
- Переключатель языка EN / RU в заголовке и внутри окна справки
- Кнопка справки (?) с руководством пользователя и переключателем языка
- SVG-логотип с изображением замка слева от заголовка страницы
- Версия приложения в футере

### Функционал
- **Вкладка «Пароль»**: слайдер длины (4–64), переключатели наборов символов (заглавные, строчные, цифры, специальные), исключение похожих символов (I, l, 1, O, 0), поля доп. символов и исключений, 8 пресетов для платформ (PostgreSQL, MySQL, SQL Server, Linux User, Windows Admin, Redis, RabbitMQ, High Entropy), индикатор надёжности (шкала 1–5 с цветовой кодировкой)
- **Вкладка «Фраза»**: слайдер количества слов (2–10), опции «С заглавной» / «Случайный регистр» (взаимоисключающие), добавление цифры в начало и/или конец, выбор разделителя (пробел, дефис, точка, подчёркивание, случайный символ, без разделителя), встроенный словарь ~500 слов
- **Вкладка «PIN»**: слайдер длины (4–12 цифр)
- Показ / скрытие результата на всех трёх вкладках
- Кнопка копирования в буфер обмена на всех трёх вкладках (текст сбрасывается через 2 с)
- Кнопка «Пересоздать» на всех трёх вкладках
- Криптографически стойкая случайность через `window.crypto.getRandomValues()` с rejection sampling для устранения смещения по модулю
